# PokéDmgCalc

Pokémon damage calculator with team and opponent tracking.

**Stack:** Vanilla JS · Supabase (auth + Postgres) · GitHub Pages

**Accounts:** Admin-created only. Sign in with your email and password.

**Features:**
- Gen 1–9 damage calculator using @smogon/calc
- Per-account My Teams and Opponent Teams tracking
- Per-account Opponents roster with team history
- Pokepaste import, type coverage charts, Smogon sets
